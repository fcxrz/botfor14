# Бот-помощник для отношений

## ВАЖНО!
Бот работает как админ-пользователь. В данном боте это не является проблемой, т.к он сделан для личного пользования: см. [планы доработки бота](#title1) 

## Функции бота для пользователя
😤 Дать пинка 😤 - даст партнеру понять, что ты готов(а) продолжить диалог, но ему(ей) нужно уступить и начать диалог.  
✨ Тёплый импульс ✨ - дать партнеру понять, что ты рядом. При желании есть возможность написать сообщение.  
🧣 Мягкий мост 🧣 - кнопка для отложенного сообщения с эмоцией. После выбора эмоции предлагается описать ее для партнера. Далее выбирается дата, когда получатель сможет посмотреть эту 'капсулу'.  
🤭 Игривый вызов 🤭 - если вы не придумали чем бы вам заняться, то этот бот предложит вам 1 идею, как вместе скоротать вечер. За основу он берет время и погоду в Омске (в коде можете поменять API на ваш город и переписать промпты).  
🩸 Сообщение Хеллсинга 🩸 - сообщение Шрёдингера. Работает с ограничением максимум в год, используется Московское время.  
🌌 Мост понимания 🌌 - очень практичная вещь! При ссорах он слушает сначала твою точку зрения, потом отправляется к партнеру, слушает его точку зрения и придумывает компромисс.  
🔔 Я в порядке 🔔 - кнопка для того, чтобы оповестить партнера, что все впорядке. Нажимать на нее можно как просто так, так и по запросу.  

## Функции бота для админа
/пасхалка - временно нерабочая команда.  
/отклик - просьба пользователя откликнуться, нажав на кнопку 🔔 Я в порядке 🔔  
/моменты - временно нерабочая команда.  
/импульс - команда-альтернатива ✨ Тёплый импульс ✨. Работает ровно так же, как и у пользователя.  

# <a id="title1">Будущее бота</a>
### МИНИ-ИГРА В БОТЕ!
Концепт: выращивание, фарм, улучшение.

### Приятные моменты
1. Бот принимает гс, файл, фотку, песню и т.п вариации сообщении 
2. Запрашивает как назвать
3. Выгружает их в гугл диск 
4. При запросе - выдает название и сами файлы.

### Квиз «Наши инсайды»:
Каждую субботу бот задает вам обоим вопрос про вашу неделю. «Что самое странное ты видела сегодня в метро?». В конце месяца бот генерирует PDF-газету вашей жизни.
